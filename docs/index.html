Architecture Course

קורס ארכיטקטורת מערכות מודרניות

תכנון ובנייה של מערכות תוכנה סקיילביליות — מ-Docker ועד CI/CD

📋 סילבוס

01 · LESSON

🐳 Docker

קונטיינריזציה, Dockerfile, Docker Compose, רשתות ו-volumes. הבנת ההבדל בין קונטיינר למכונה וירטואלית.

02 · LESSON

⚡ Caching

עקרונות cache, Redis, Cache-Aside / Write-Through, TTL ו-invalidation לשיפור ביצועים.

03 · LESSON

🧩 Microservices

מעבר ממונוליט, תקשורת sync/async, Bounded Context ועקרונות Domain Driven Design בסיסיים.

04 · LESSON

🚪 API Gateway

Single entry point, routing, authentication, rate limiting וניהול ריכוזי של תקשורת מול שירותים.

05 · LESSON

⚖️ Load Balancer

Round Robin, Least Connections, IP Hash, Health Checks, High Availability ו-Horizontal Scaling.

06 · LESSON

🐰 RabbitMQ · תורים

Producer/Consumer, Exchanges, Dead Letter Queues, Acknowledgments ו-async messaging לניתוק שירותים.

07 · LESSON

🔄 CI/CD

Continuous Integration & Deployment, GitHub Actions, בדיקות אוטומטיות וניהול סביבות dev/staging/production.
